# VSCode 快捷键
1. WSL 目录在 VSCode 中打开：进入相应目录，输入 `code .` 就可以在 VSCode 中打开该目录；
2. 打开终端： `CTRL + \``;
3. 打开插件中心： `CTRL + SHIFT + X`;
4. 调整窗口大小： `CTRL + +` 放大窗口（不仅是编辑器字体）；`CTRL + -`缩小窗口；
5. 进入全屏模式：`F11`,全屏窗口会隐藏顶部菜单栏；
6. 打开命令面板：`CTRL + SHIFT + P`;
7. 查找目录下的文件: `CTRL + P`。
